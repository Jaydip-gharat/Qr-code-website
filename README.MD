# Frontend Mentor - QR Code Component Solution

This is my solution to the Frontend Mentor QR Code Component challenge. The goal of this project was to recreate a QR code card component using HTML and CSS while practicing layout, alignment, responsive design, and CSS custom properties.

### What I Learned

While building this project, I practiced:

* Centering elements both vertically and horizontally using Flexbox.
* Creating reusable color variables with the `:root` selector.
* Structuring a simple card component with HTML.
* Using media queries for responsive layouts.
* Applying shadows and border radiuses to create a modern card design.

Example of CSS variables used in the project:

```css
:root {
  --White: hsl(0, 0%, 100%);
  --Slate300: hsl(212, 45%, 89%);
  --Slate500: hsl(216, 15%, 48%);
  --Slate900: hsl(218, 44%, 22%);
}
```

Example of Flexbox centering:

```css
body {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```
